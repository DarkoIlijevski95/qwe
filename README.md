##1. Опис на апликацијата
Оваа апликација претставува еден облик на најдолго емитуваниот квиз на Балканот, "ТБ Слагалица". Во овој квиз има 5 различни игри: Погоди, Математика, Прашање, Спојувалка и Асоцијација.
